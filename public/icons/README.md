This directory holds PWA icons for CargoExpress PH.

Required icon files:
- icon-72.png    (72x72)
- icon-96.png    (96x96)
- icon-128.png   (128x128)
- icon-144.png   (144x144)
- icon-152.png   (152x152)
- icon-192.png   (192x192)
- icon-384.png   (384x384)
- icon-512.png   (512x512)
- icon-maskable-192.png  (192x192, with safe zone padding)
- icon-maskable-512.png  (512x512, with safe zone padding)

Generate from public/favicon.svg using a tool like:
  npx pwa-asset-generator public/favicon.svg public/icons
